ExpLung-U is a deep learning-based diagnostic platform designed to enhance chest scan radiology. Developed as a split-stack application using Next.js and FastAPI, the system integrates custom-trained DenseNet-121 classification heads, neural explanations (SmoothGradCAM++), thoracic Attention U-Net segmentations, and WebGL-driven 3D volumetric lung reconstructions.

By targeting the convolutional layers of our classifier, the SmoothGradCAM++ pipeline extracts pixel-specific warmth heatmaps directly overlaying raw X-Ray files, highlighting anomalous pulmonary consolidation regions for clinicians. Concurrently, our thoracic segmenter isolates active lung boundaries, computing left and right lung involvement percentages to automatically map scans onto graded medical severity charts.

To provide radiologists with an interactive spatial cockpit, the backend converts 2D segmented lung contours into 3D organ voxel meshes. Utilizing ellipsoidal coordinate boundaries and tapering depth equations, the marching cubes algorithm synthesizes dual-mesh GLB files. These meshes allow clinicians to interactively rotate, pan, and toggle views between clean anatomy and crimson warmth anomalies using Three.js inside the browser. To optimize performance, the frontend dynamically imports WebGL packages on-demand, maintaining rapid loading layouts.

Furthermore, ExpLung-U includes a clinician notes interface that seamlessly syncs inputs to a remote Supabase instance. If schema constraints occur, dynamic pruning handles exception fallbacks automatically. A dynamic PDF engine generates clean, printable A4 clinical report cards featuring patient demographics, segmented severity grids, clinician notes, and AI-supportive care instructions with OTC medication suggestions.

Developed by Riya, Swaym, Vedant, and Yash, ExpLung-U demonstrates how custom deep learning, advanced coordinate mathematics, and secure cloud pipelines can combine to bridge the cognitive gap in digital diagnostic workflows. This project serves as a highly robust, explainable, and responsive medical CAD prototype ready for clinical presentation.
